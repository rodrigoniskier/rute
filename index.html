Bem-vindo ao Estudo Interativo de Rute

Esta página oferece uma análise gramatical detalhada de cada palavra do livro de Rute, baseada no texto hebraico.

Para começar:

  1. Escolha um capítulo no menu à esquerda.
  2. O texto do capítulo aparecerá aqui. Clique em um versículo.
  3. Uma lista de palavras aparecerá no menu à direita.
  4. Clique em uma palavra para ver sua análise completa.
